DICAS E REVISÃO DE PRINCIPAIS COMANDO GIT

Git clone + url

git log -> mostra todos detalhes de quem é autor hora etc do commit.

git log --oneline -> mostra os commits apenas descritos por sua descrição e hash.

git commit app.js -m "alterando app.js..." -> commit indicando o nome do arquivo que quero alterar.

git commit . -m "alterando multiplos arquivos" -> altero multiplos arquivos apenas com um commit.(geralmente será o usado... ele altera o diretorio atual e todos de dentro.)

git push -> envia as modificações.(pega todas modificações, todos commits, e manda bala no repositório)

git restore --source c776f0cde -> indica o ponto em que queremos retornar nosso código(desfazer uns commits, voltar a forma anterior)

git checkout -b nomeDaBranch -> cria uma nova branch para trabalhar em paralelo ao código principal

git branch nome-da-branch -> também cria uma nova branch. porém não muda automaticamente para ela, pra isso precisa usar o comando git switch

git branch -> mostra todas as branch existentes no projeto e já mostra a branch que eu to

git switch escolheABranch -> vai para a branch escolhida.
